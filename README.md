Sordid Dragon
=============

jQuery plugin for vertical sorting

Tested on:
* Chrome
* Firefox
* IE8-11
* Safari
* Android
* iOS
* Microsoft Surface


Dependencies
============
jQuery `$.browser` (moved to jQuery Migrate in 1.9)
